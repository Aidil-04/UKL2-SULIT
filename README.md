# Program Penjumlahan Matriks
Program Penjumlahan Matriks digunakan untuk melakukan penjumlahan dua buah matriks berdimensi sama berdasarkan input dari pengguna.
CARA KERJA:
- Program menggunakan Scanner untuk membaca input dari keyboard.
- Pengguna memasukkan jumlah baris dan kolom.
- Program membuat tiga matriks:
- matriks1 → matriks pertama
- matriks2 → matriks kedua
- hasil → matriks hasil penjumlahan
- Elemen-elemen dari matriks1 dan matriks2 diisi melalui input pengguna.
- Program menjumlahkan elemen-elemen yang bersesuaian dari kedua matriks:
\mathrm{hasil}[i][j]=\mathrm{matriks1}[i][j]+\mathrm{matriks2}[i][j]- Matriks hasil ditampilkan ke layar.
